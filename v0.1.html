Quick start

CoderStyle is syntax highlighting project for web developper , very easy to include and integrate in your awesome web application .
Last version : v0.1
Languages supported : html , css
Templates available : Monokai , Eiffel , Solarized , Twilight

Install

Very easy to include in your application , just download or clone from github

git clone https://github.com/marwenhlaoui/CoderStyle.git

Using

You will need to include the template cs-monokai.min.css and the scriptcoderstyle.min.js files you downloaded or by cdn in your layout or pages.
